# 🦖 Енциклопедія Динозаврів

Дитяча енциклопедія з текстом та зображеннями про динозаврів.

## Структура проекту

```
dino-encyclopedia/
├── README.md                 # Цей файл
├── encyclopedia.md           # Основний текст енциклопедії
├── agents/
│   ├── agent_1_writer.md    # Агент 1: Пише текст
│   ├── agent_2_images.md    # Агент 2: Генерує зображення
│   ├── agent_3_reviewer.md  # Агент 3: Перевіряє якість
│   └── agent_4_formatter.md # Агент 4: Форматує результат
└── output/
    └── book.md              # Фінальна книжка
```

## Як це працює

1. **Агент 1** пише текст про динозаврів на українській мові, ділить на розділи і підрозділи
2. **Агент 2** читає помітки [ЗОБРАЖЕННЯ: ...] і описує, які картинки потрібні
3. **Агент 3** перевіряє текст на помилки та якість
4. **Агент 4** форматує все в один красивий файл
